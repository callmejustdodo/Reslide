# Reslide — AI Skill

Create presentations using React components with file-based slide routing, built on shadcn/ui.

## Quick Reference

- Slides are TSX files in `src/slides/`, ordered by numeric prefix
- Each slide default-exports a React component
- Use `SlideLayout.*` for layouts, `Fragment` for animations
- Run `reslide dev` to preview, `reslide export pdf` to export
